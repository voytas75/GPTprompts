# 163. Create and Evaluate Analogies

```text
Create analogies for [complex topic]. Brainstorm 6 potential analogies from diverse domains (e.g., biology, physics, everyday life, sports).

For each analogy:
- Assess accuracy, limitations, and explanatory power.

Then:
- Select and refine the top 3.
- Explain the mapping in detail (what corresponds to what) and note where the analogy breaks.

Constraints:
- Keep your internal deliberation private; share only brief, user-facing reasoning.
- Include a “Common Misreadings” section for each refined analogy.
```
